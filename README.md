## 安装

1. 通过`pip`或`nb`安装，版本请指定`0.1.2`；

2. 在`env`内设置：

```python
ANTI_FLASH_ON=true                          # 开启或关闭
ANTI_FLASH_GROUP=["123456789", "987654321"] # 指定群聊
```

确保打开功能时群聊列表不为空。

## 功能

⚠ **谨慎开启此项功能, 谨慎指定群聊**

由于该功能过于危险，需指定特定群聊启用反闪照功能。
